# Contributing to PreeclampsiaRisk · WVSUMC

Thank you for your interest in this project.

**PreeclampsiaRisk** is a proprietary research prototype developed for West Visayas
State University Medical Center (WVSUMC), College of Medicine. The codebase is
closed to external feature contributions and pull requests.

---

## What Is Accepted

### Bug Reports

Bug reports are welcome and genuinely helpful. If you encounter a reproducible
technical issue — broken layout, incorrect calculation output, print failure,
or browser-specific behavior — please open a GitHub Issue using the
**Bug Report** template.

A good bug report includes:

- A clear, specific title
- Steps to reproduce the issue (exact values entered if calculation-related)
- Expected behavior vs. actual behavior
- Browser name and version
- Device type (desktop / mobile / tablet)
- Screenshots or screen recordings if applicable

---

## What Is Not Accepted

The following will be closed without review:

- Pull requests of any kind
- Feature requests via Issues (use the correct template)
- Requests to open-source or relicense the project
- Issues reporting disagreement with clinical scoring weights
  *(weights are literature-derived proxies pending study completion — this is intentional)*
- Duplicate issues already documented in [`CHANGELOG.md`](./CHANGELOG.md)

---

## Clinical Feedback

If you are a **clinician or researcher** and have feedback on the scoring
methodology, parameter thresholds, or clinical validity of the output, please
contact the owner directly rather than opening a public Issue.

Contact: [gdddevelopers.dev@gmail.com](mailto:gdddevelopers.dev@gmail.com)

---

## Code of Conduct

All interactions in this repository are expected to be professional and
respectful. Issues that are abusive, off-topic, or submitted in bad faith
will be closed and the user reported to GitHub.

---

## Scope

This tool is a **research prototype**. It is not a certified medical device.
Issues relating to clinical decision-making or patient outcomes are outside
the scope of this repository entirely.
